Deploy to GitHub Pages (Free)

Since this converter is built purely as a static client-side web application, you can publish it live on your GitHub Repository for instant access on any device with these easy steps:

  1. Name this HTML code file exactly index.html.
  2. Commit and push index.html directly to your main repository branch.
  3. In GitHub, click on your repository's Settings tab.
  4. On the left sidebar, select Pages.
  5. Under Build and deployment, set Source to Deploy from a branch.
  6. Choose your branch (typically main) and folder path (/ (root)), then click Save.

Within 1-2 minutes, GitHub will issue a dynamic web URL (e.g. https://username.github.io/repo-name/) where you can access and drag files to this converter from any phone, laptop, or tablet!
